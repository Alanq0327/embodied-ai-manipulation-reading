# embodied-ai-manipulation-reading
# Embodied AI and Learning-Based Robot Manipulation Reading Project

This repository contains an independent reading report on language-grounded planning and learning-based manipulation for embodied robots. The project focuses on how recent advances in large language models, vision-language models, and generative robot policies can be connected with executable robot behaviors.

The selected papers cover several connected topics, including language grounding, open-world task planning, functional grasping, diffusion-based visuomotor policy learning, and physically feasible bimanual manipulation.

## Motivation

Embodied AI systems require more than high-level reasoning. A robot must be able to connect language, perception, planning, action generation, and physical execution. This reading project aims to understand how semantic reasoning from language or vision-language models can be grounded into robot affordances, grasping strategies, motion policies, and control constraints.

My background includes embedded robotics, state estimation, sensor fusion, real-time control, multi-agent decision-making, and robot learning. In previous projects, I worked on autonomous mobile robot control, IMU/ToF-based sensing, Kalman-filter-based estimation, PID control, mapping, localization, BLE-based robot logging, and cooperative task planning in simulated embodied environments.

This reading project serves as a bridge between my prior hands-on robotics experience and broader research directions in embodied AI and robot manipulation.
